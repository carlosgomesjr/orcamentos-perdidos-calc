# Calculadora de Orçamentos Perdidos

Ferramenta standalone para estimar vazamento de oportunidades em lojas de persianas, cortinas e decoração sob medida.

## Arquivos

- `index.html`: calculadora pronta para publicar.
- `assets/`: ativos visuais usados pela calculadora.
